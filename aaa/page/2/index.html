i3与dolphin文件关联

发表于 2024-06-14 更新于 2024-08-19

错误说明

  • 新版本的kde5下,i3桌面下dolphin无法正确使用文件关联

解决方案一（最佳方法）

  1. 安装downgrade降级软件
  2. 降级 kservice5,kservice5 5.115.0-1
  3. 不要选择忽略kservice5升级
  4. 降级后, copy applications.menu from /etc/xdg/menus/ to ~/.config/menus/
  5. 重新升级kservice5

解决方法二（不太好的方法,方法一失败时使用）

  1. 下载plasma-applications.menu
  2. 将其复制到.config/menus/和/etc/xdg/menus下并重命名为applications.menu

参考文献

  • https://forum.endeavouros.com/t/i3-issue-with-dolphin-and-file-associations-in-i3/54405/4

    1. 如果在同一目录下有多个源文件，需要在add_executable里把所有源文件都添加进去，这太不优雅了。因此使用aux_source_directory(dir var)把指定目录下所有的源文件存储在一个变量中，第一个参数dir是指定目录，第二个参数var是用于存放源文件列表的变量名。

    2. 当程序文件比较多时，我们会进行分类管理，把代码根据功能放在不同的目录下，这样方便查找。这个时候头文件定位需要用以下命令来实现，不够优雅。

      因此使用include_directories 方法添加多个指定头文件的搜索路径，路径之间用空格分隔。

    3. 源代码随意放置，不够优雅，统一放入src文件夹后要用add_subdirectory(source_dir [binary_dir] [EXCLUDE_FROM_ALL])设置源代码目录。其中**source_dir 指定一个包含CMakeLists.txt和代码文件所在的目录;binary_dir二进制代码目录执行后的输出文件将会存放在此处;EXCLUDE_FROM_ALL标记，表示新增加的子目录将会排除在ALL目录之外。**

samba配置

发表于 2023-02-02 更新于 2024-09-06

Arch

  1. 安装samba
    paru -S samba
  2. 创建配置文件
    sudo vim /etc/samba/smb.conf
  3. 写入需要共享的文件夹
  4. 创建samba日记文件夹:
    sudo mkdir -p /usr/local/samba/var/
  5. 添加samba 访问用户并设置密码，smb_user为你自己设置的用户名：
    sudo smbpasswd -a smb_user
  6. 启动服务
    sudo systemctl enable --now smb.service nmb.service
  7. 在win10文件夹点击网络，输入“\xxx”,xxx为linux宿主机的ip地址

win10

  1. 打开启用或关闭Windows功能，勾选NFS服务和SMB1.0/CIFS文件支持，重启
  2. win+r，在运行框输入“gpedit.msc”，enter回车，进入组策略编辑器。
  3. 在组策略编辑器中找到“计算机配置”–“管理模板”–“网络”–“Lanman工作站”。点击“启用不安全的来宾登录”

vim常用快捷键

发表于 2022-12-22 更新于 2024-09-06

你想以最快的速度学习人类史上最好的文本编辑器VIM吗？你先得懂得如何在VIM幸存下来，然后一点一点地学习各种戏法。

存活

  • 了解normal模式和insert模式,在normal模式下按i进入insert模式,按ESC回到normal模式
  • h/j/k/l在normal模式下分别表示向左下上右移动
  • :q!不保存退出
  • :w保存

进阶

快速移动

  • gg/G移动到文件的开头和结尾,可以用ctrl+o快速返回
  • H/M/L跳转到屏幕的开头(HEAD),中间(Middle)和结尾(Lower)
  • ctrl+d(down)往前翻半页,ctrl+u(up)往后翻半页
  • ctrl+f(forward)往前翻一页,ctrl+b(backward)往后翻一页
  • zz让光标所在行居屏幕中央
  • zt让光标所在行居屏幕最上一行 t=top
  • zb 让光标所在行居屏幕最下一行 b=bottom
  • w/W移到下一个word/WORD开头,e/E下一个word/WORD尾
  • b/B(backword)回到上一个word/WORD开头
  • 使用f{char}可以移动到char字符上,t移动到char的前一个字符,用分号(;)/逗号(,)继续搜索该行下一个/上一个,3f{char}可以移动到第三个char字符上
  • 0移动到行首,$移动到行尾
  • 字符# 和 * : 匹配光标当前所在的单词，移动光标到下一个（或上一个）匹配单词（*是下一个，#是上一个）
  • % : 匹配括号移动，包括 (, {, [

增删改查

  • dw删除单词,x删除字符
  • dt)删除右边到)为止的所有字符，其中”)”可以为任意字符
  • r(replace)可以替换一个字符
  • s(substitute)替换并进入插入模式
  • c(change)配合文本对象快速修改,例如caw先回到当前单词光标首字符并删除单词并进入插入模式,cw从当前光标删除到单词结束并进入插入模式,ct)删除到括号并进入插入模式,其中”)”可以为任意字符
  • 使用/或者?进行前向或者反向搜索
  • 使用n/N跳转到下一个或者上一个匹配
  • 使用*或者#进行当前单词的前向或者后向匹配

wintogo&&linuxtogo

发表于 2022-06-21 更新于 2024-08-19

ventoy安装制作u盘

  1. 找个固态u盘或者移动固态,制作成ventoy启动盘(略)
  2. 将ventoy的启动盘格式化成NTFS格式
  3. 安装vhd文件启动插件，放在U盘第1个分区（就是放ISO文件的分区）的 ventoy 目录下。vhd文件启动插件

virtualbox安装linux或windows镜像系统

安装和普通流程基本一样,主要讲两点不太一样的

  1. 虚拟硬盘文件类型选择专家模式,选择VHD虚拟硬盘,选择固定大小
  2. 在硬盘创建后还未启动时开启UEFI,设置–系统–扩展特性–启用EFI

linux镜像系统添加ventoy插件

  1. 在虚拟机中打开浏览器,输入ventoy.net,选择中文,选择文档手册,选择插件说明,选择Linux vDisk启动插件,于”在系统下执行vtouboot脚本”处下载插件
  2. 将插件解压,以root模式安装sudo sh vtoyboot.sh,出现vtoyboot process successfully finished即成功

linuxtogo提取制作

  1. 将linux虚拟机关机,打开虚拟机安装目录,找到.vhd文件,在后面加上.vtoy,移动到ventoy启动盘即可
  2. linux内核升级后建议重新执行sudo sh vtoyboot.sh

wintogo提取制作

  • 将windows虚拟机关机,打开虚拟机安装目录,找到.vhd文件,移动到ventoy启动盘即可

wintogo扩容

  1. 查看虚拟磁盘wtg.vhdx信息
    qemu-img info wtg.vhdx
  2. vhdx格式转换成qcow2
    qemu-img convert -f vhdx -O qcow2 wtg.vhdx wtg.qcow2
  3. 对wtg.qcow2扩容到100G
    qemu-img resize wtg.qcow2 100G
  4. 将qcow2文件转换回vhdx格式
    qemu-img convert -f qcow2 -O vhdx wtg.qcow2 wtg_expanded.vhdx
  5. 重命名文件
    mv wtg.vhdx wtg_old.vhdx
    mv wtg_expanded.vhdx wtg.vhdx
  6. 最后进入虚拟硬盘中,利用windows的磁盘管理工具扩容C盘

home目录迁移方法

发表于 2022-04-25 更新于 2024-08-19

查看硬盘uuid

sudo blkid

开机自动挂载硬盘,在文件的最后新增加一行代码

sudo vim /etc/fstab
输入硬盘UUID /media ext4 defaults 0 0

临时挂载硬盘

sudo mount /dev/sdb1 /media

创建软链接

sudo ln -s /media/abelxiaoxing /home/abelxiaoxing

创建用户,并制定用户目录

sudo useradd -d /home/abelxiaoxing abelxiaoxing

修改目录所属权

sudo chown -R abelxiaoxing:abelxiaoxing /media/abelxiaoxing
