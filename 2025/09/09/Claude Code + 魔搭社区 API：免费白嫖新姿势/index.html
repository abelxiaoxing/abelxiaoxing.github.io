Claude Code + 魔搭社区 API：免费白嫖新姿势

最近发现魔搭社区（ModelScope）已经支持 Anthropic API，这意味着我们可以通过一些简单的配置，让 Claude Code 直接使用魔搭社区提供的免费 API 资源。亲测可用，这里给大家分享一下具体步骤。 ## 核心思路

核心原理非常简单：通过修改环境变量，将 Claude Code 的 API 请求地址指向魔搭社区的 API 端点。 ## 魔搭社区API

  1. 打开魔搭社区：https://modelscope.cn/models
  2. 筛选可以api调用的模型：点击支持体验–>推理 APl-Inference,支持通过标准推理 AP接口迅速调用
  3. 点击推理 API-Inference下的Anthropic查看代码示例，复制自己的apikey和对应的模型ID。

claude code安装

  1. 安装pnpm 推荐使用 pnpm 而不推荐 npm，因为 pnpm 的安装速度更快，并且通过符号链接的方式能更有效地管理依赖，占用磁盘空间更小。关于 pnpm 的具体安装过程这里不再赘述，可以参考其官方文档进行安装。
